# Frontend Development Laboratory

This repository contains all laboratory exercises for Frontend Development course.

## Structure

### Laboratory 01 - HTML
- **lab-01-holy-grail/** - Holy Grail Layout (3-column layout without Flex/Grid)
  - Classic blog layout: Header | (Navigation | Content | Sidebar) | Footer
  - Uses: position, float, negative margins

### Laboratory 02 - CSS

#### Task 1: Product Page
- **lab-02-product-page/** - Responsive e-commerce product page
  - Flexbox layout
  - Media queries (3 breakpoints)
  - Hover effects and animations
  - Example: Allegro/Amazon style

#### Task 2: Navbar
- Included in lab-02-product-page/
  - CSS Grid navigation
  - Animated hamburger menu
  - Slide-in overlay on mobile
  - Example: Decathlon style

#### Task 3: Dashboard
- **lab-02-dashboard/** - Statistics dashboard
  - CSS Grid with 6 tiles
  - CSS Custom Properties (variables)
  - Light/Dark theme toggle
  - Hover effects (lift, shadow)
  - Example: Fitness/Gaming apps (Diablo stats)

#### Bonus: Player Profile
- **lab-02-player-profile/** - Player statistics profile
  - Grid layout
  - Stats cards with progress bars
  - Recent matches table
  - Responsive design

## How to Run

Open any `index.html` file in a browser.

Each folder is self-contained with all necessary files (HTML, CSS, JS, assets).

## Technologies Used

- HTML5 semantic elements
- CSS3 (Flexbox, Grid, Custom Properties, Transitions)
- Vanilla JavaScript (for interactions)
- No external frameworks (pure HTML/CSS/JS)

## Changes Made

Recent refactoring to reduce AI-generated aesthetics while maintaining all lab requirements:
- Removed excessive letter-spacing
- Reduced border-radius (16px → 3-4px)
- Removed gradient backgrounds and glows
- Changed fonts from Inter to Roboto
- Simplified color schemes
- Replaced SVG placeholders with real images
- Reduced excessive whitespace

All technical requirements (Flexbox, Grid, media queries, animations, CSS variables) are preserved.
