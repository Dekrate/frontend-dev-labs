# Laboratory 02

## Task 1

Responsive e-commerce product page using **Flexbox** and **media queries**.

## How to run

- Open `index.html` in a browser.
- Resize the viewport:
  - Mobile: up to `768px` → stacked layout
  - Desktop: from `769px` → split layout (image + info)

## Task 2

Animated sticky navbar with hamburger menu overlay using **CSS Grid**, transforms, transitions, and hover/focus states.

## How to run

- Open `index.html` in a browser.
- Navbar breakpoints:
  - Mobile: `<768px` → hamburger + slide-in overlay menu
  - Tablet: `768–1024px` → full links (no hamburger)
  - Desktop: `>1024px` → full links (wider spacing)
